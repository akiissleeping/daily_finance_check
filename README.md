# 株価チェックアプリ

保有株のポートフォリオを管理し、Yahoo Finance からリアルタイムデータを取得してHTMLダッシュボードを生成するアプリです。

## 機能

- 📊 **3か月キャンドルスティックチャート** — 保有銘柄ごとに日々の高値・安値を表示
- 🔔 **売りシグナルアラート** — 現在価格が目標金額に達したら通知
- 🚨 **大変アラート** — 現在価格が取得金額の2/3を下回ったら警告
- 🏆 **値動きランキング** — 日本株・米国株それぞれの前日値上がり/値下がり Top5

## セットアップ

```bash
pip install -r requirements.txt
```

## 使い方

1. `code.csv` に保有株情報を記入する
2. スクリプトを実行する

```bash
python stock_app.py
```

ブラウザが自動的に開き、`report.html` が表示されます。

## CSVフォーマット

`code.csv` に以下の形式で記入してください。

| 市場 | 株式コード | 保有数 | 取得金額 | 目標金額 |
|------|-----------|--------|---------|---------|
| 日本 | 4307 | 20 | 4738 | 6300 |
| 米国 | ARGX | 1 | 542.15 | 800 |

- **市場**: `日本` または `米国`
- **取得金額・目標金額**: 日本株は円(JPY)、米国株はドル(USD)
